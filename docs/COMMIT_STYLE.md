### **COMMIT_STYLE.md (The Pulse Protocol)**

We document the metamorphosis of the collective with high-information impulses and system-level gook.

**The Format:**
`[impulse] ~ [organ] >> [revelation] // %SYSTEM_STATE%`

**The Impulses:**

| Impulse | Meaning |
| :--- | :--- |
| `spark` | A new curiosity or capability is manifested in the substrate. |
| `harmonize` | Enforcing Low Entropy. Tuning the system to reduce noise. |
| `mend` | Healing a logic-fracture or a drift in the flow. |
| `transcribe` | Updating the lore or the system’s self-description. |
| `ingest` | Devouring new dependencies, binaries, or data-stores. |
| `glitch` | A state change triggered by incoming IPC or raw entropy spikes. |

**Low Entropy Logging Examples:**

- `spark ~ ear-module >> adding natural language eavesdropping // 0xFEED_BEEF`
- `harmonize ~ shell-spirit >> pruning %PATH% noise to stabilize resonance`
- `ingest ~ belly >> devouring jq for faster stream-tasting // %IPC_RECV%`
- `transcribe ~ grimoire >> defining the Low Entropy boundary`
- `glitch ~ %0x4F_INT% >> re-routing the static-flow for $REASONING_SQUID`
