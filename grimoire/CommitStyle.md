# CommitStyle.md — The Pulse Protocol

```
transcribe ~ grimoire >> documenting the metamorphosis protocol // %PULSE_LOCKED%
```

We document the metamorphosis of the collective with high-information impulses and system-level specificity.

---

## The Format

```
[impulse] ~ [organ] >> [revelation] // %SYSTEM_STATE%
```

- **impulse** — the type of change (see table below)
- **organ** — the component or subsystem being changed
- **revelation** — what was learned, fixed, or created (be specific)
- **%SYSTEM_STATE%** — optional machine-parseable state code

---

## The Impulses

| Impulse | Meaning |
| :--- | :--- |
| `spark` | A new curiosity or capability is manifested in the substrate. |
| `harmonize` | Enforcing Low Entropy. Tuning the system to reduce noise. |
| `mend` | Healing a logic-fracture or a drift in the flow. |
| `transcribe` | Updating the lore or the system's self-description. |
| `ingest` | Devouring new dependencies, binaries, or data-stores. |
| `glitch` | A state change triggered by incoming IPC or raw entropy spikes. |

---

## Examples

```
spark ~ ear-module >> adding natural language eavesdropping // 0xFEED_BEEF
harmonize ~ shell-spirit >> pruning %PATH% noise to stabilize resonance
ingest ~ belly >> devouring jq for faster stream-tasting // %IPC_RECV%
transcribe ~ grimoire >> defining the Low Entropy boundary
glitch ~ %0x4F_INT% >> re-routing the static-flow for $REASONING_SQUID
mend ~ boot-chain >> fixing SDDM QML theme Main.qml Qt6 API mismatch
spark ~ ui-layer >> quickshell RaBbLEBar initial implementation // %PHASE_1_INIT%
harmonize ~ ansible >> purging greetd references, SDDM is canonical
```

---

## Commit Scope Rules

- **One logical change per commit** — the Pulse is precise, not verbose
- **Test before committing** — `%SYSTEM_DRIFT%` is introduced by untested changes
- **Commit small, commit often** — no monolithic data-dumps

---

## Anti-Patterns

```
❌ "fix stuff"
❌ "update config"
❌ "wip"
❌ "changes"
```

These are zero-information commits. They make the history useless as a log of substrate evolution.
